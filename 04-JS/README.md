## Desafio de JS

El proyecto  debe contar con funcionabilidades:

* Lista de pokemons
* Diseño de pokedex
* Utilizar poke-API
* Mostrar nombre, atributos y clases
<br>

* Metodología BEM CSS
* Vite
* JavaScript
* Libreria Axios

### 💻 Listados comandos 

* `npm install `  instalara las dependencias de vite
* `npm run dev `  iniciara el servidor para vista previa en localhost:5173
